# 🗂 Task Manager App

A full-stack task management application built with React, Node.js, Express, and SQL Server.

The application allows users to securely register, log in, and manage their personal tasks. Users can create, update, and delete tasks while tracking priorities, deadlines, and completion status.

Additionally, the application includes a secure password recovery system that sends reset codes via email.

This project demonstrates full-stack development, including frontend UI design, backend API creation, authentication, database integration, and email services.

---

# 🚀 Features

## Authentication

- User registration  
- Secure login with JWT authentication  
- Password hashing using bcrypt  
- Protected API routes  

---

## 🗂 Task Management

Users can manage tasks through a dashboard:

- Create new tasks  
- Edit existing tasks  
- Delete tasks  
- Set priority levels  
- Add deadlines  
- Track task status  

### Task statuses include:

- Not Started  
- In Progress  
- Completed  

---

## 📊 User Dashboard

After logging in, users are redirected to a dashboard where they can:

- View all their tasks  
- Add new tasks  
- Update task details  
- Delete tasks  
- Track deadlines and priorities
